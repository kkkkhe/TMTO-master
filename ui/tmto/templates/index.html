What is TMTO ??

Time-memory trade-off analyzes the trade-off between time and memory while solving a problem.
Simulations of three functionalities have been done.

  • Analyse m-t - A 3d plot depicting observed P(S) for different values of ‘m’ and ‘t’.
  • Iterate - Gives the probability of success P(S) for the given parameters
  • Single Attack - Search for the key using algorithm.
